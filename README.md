##  💭  Background
### 1. 연애 목적의 친구를 찾는 목적에만 치중된 친구매칭 서비스 시장
연애 목적의 친구가 필요한 게 아닌데, 그냥 순수하게 우리만 아는 학교 얘기를 나눌 수 있는 같은 학교 친구, 수업을 듣다가 모르는 내용을 물어볼 수 있는 같은 학과 친구, 언제든 불러내 카페에서 같이 공부할 수 있는 동네 친구, 공모전에 같이 나갈 수 있는 팀원들을 만나고 싶어요!
### 2. 팀원 구인 과정 속 일회성 의사소통 수단의 불가피한 개인정보 노출
팀원을 모집하려는데 일회성 의사소통을 하려면 카카오톡 같은 타 커뮤니케이션 어플리케이션을 이용해야 해요. 그래서 의도치 않은 개인정보(이름, 사진 등)가 노출될 수 있고요. 대화 끝에 추구하는 방향이 달라 팀원으로 함께하지 못하게 된다면.. 모르는 유저에게 개인정보만 보여준 셈이 되는걸요.
### 3. 팀원의 이전 단체활동 참여 태도 파악 불가
유저 본인이 작성한 주관적인 소개만 확인할 수 있기 때문에, 이전 단체활동에서 해당 유저가 어떻게 참여했는지 참여 태도를 파악하기 어려워요. 중간에 흐지부지되는 일 없이 성실한 팀원과 함께 하고 싶은데 알 방법이 없어요.
### 4. 팀원 구인을 위해 여러 어플리케이션을 거쳐야 하는 피로감
로켓으로 공고를 확인하고, 캠퍼스픽에서 팀원을 구하고, 카카오톡으로 넘어와서 대화하고.. 그냥 한 어플리케이션 안에서 모든 과정을 끝낼 수는 없을까요?<br><br><br>


##  📝  Features
### 1. 유저 맞춤형 친구 조회
유저의 설정 정보에 따라 원하는 분야별로 학교 친구, 학과 친구, 동네 친구를 조회할 수 있어요.
### 2. 어플리케이션 내 채팅
미모티콘으로 설정하는 프로필사진, 익명성을 보장하는 닉네임! 개인정보 노출 우려 없이 어플리케이션 내에서 채팅을 할 수 있어요.
### 3. 팀원의 참여 태도 평가 가능
팀 리더가 설정한 활동종료일에 맞춰 활동이 종료되면 팀원들의 참여 태도를 성실성, 팀워크, 기여도로 세분화하여 평가할 수 있어요. 별점의 평균은 유저의 프로필에 반영됩니다.
### 4. 팀의 생성부터 종료까지 관리
팀원 구인 글을 작성하면 다른 유저는 해당 글에 참가신청을 할 수 있습니다. 작성자는 들어온 신청을 통해 해당 유저의 프로필로 이동해 자유 양식으로 작성된 역량을 확인할 수 있습니다. 소개를 보고 함께했을 때 시너지를 낼 수 있을 것 같다면 채팅을 시작해보세요. 충분한 이야기를 나누고 팀으로 함께 하기로 결정했을 경우 참가신청을 수락하면 팀이 생성됩니다. 이후 팀 활동 종료시 팀원의 참여 태도 평가 기능까지. 드림메이트는 여러분의 팀 활동을 처음부터 끝까지 함께합니다.<br><br><br>
